Hello everyone

This is a demonstration of Three.js, a popular JavaScript library for creating 3D graphics in web applications. Three.js provides a simple yet powerful API for creating and manipulating 3D objects, applying textures and materials, setting up lighting, and animating scenes. With Three.js, developers can easily integrate 3D graphics into their web projects, ranging from simple visualizations to immersive experiences. We've utilized several key features of Three.js in this demo:

3D Modeling

I've created 3D models of a cube, Earth, and Moon using Three.js's geometry classes.

Texture Mapping

Textures are applied to the 3D models to add realism. Image textures for the Earth and Moon are loaded and applied to their respective materials.

Lighting and Shadows

Directional light sources simulate sunlight, enhancing the appearance of 3D objects and casting shadows on the scene.

Animations

Rotation animations for the cube, Earth, and Moon are implemented using JavaScript's requestAnimationFrame() method. Additionally, the Moon orbits around the Earth using trigonometry calculations.

Interactivity

User interaction is enabled by allowing resizing of the window, which automatically adjusts the aspect ratio of the camera and renderer, ensuring a responsive 3D scene.
